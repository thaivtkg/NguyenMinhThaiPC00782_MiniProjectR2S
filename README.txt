Xin chào quý công ty R2S
Em tên là Nguyễn Minh Thái hiện đang học Cao đẳng Thực hành FPT Polytechnic
Đây là mini project mà công ty đã giao cho em dựa vào các yêu cầu trong file excel,
em đã chọn đề tài Giỏ hàng service cùng với CRUD USER,login,logout.
Giỏ hàng bao gồm các chức năng:
	+Thêm sản phẩm vào giỏ hàng
	+Thay đổi số lượng sản phẩm trong giỏ
	+Xóa sản phẩm trong giỏ
	+Làm mới giỏ hàng
	+Chức năng checkout để mua hàng
	+Danh sách đơn hàng và chi tiết đơn hàng đã mua
Project sử dụng ngôn ngữ java spring boot làm backend , angularjs để sử dụng các rest api, cơ sở dữ liệu mysql, có sử dụng swagger để làm document.
Hướng dẫn sử dụng: 
-Sau khi đã clone thành công project về máy
-Chạy file scriptDatabase để tạo database và bảng ,scriptdata.sql để khởi tạo data bằng mysql
-Tiến hành import vào eclipse
-Thay đổi spring.datasource.password sang mật khẩu của mysql
-Run project vào nhập vào địa chỉ web : http://localhost:8080/index vào trang chủ và sử dụng các chức năng
Cảm ơn quý công ty đã xem.
